Tstat produces a "log_chat_complete" file which logs every Instant Messaging (MSN+Yahoo+Jabber) possible flow 
pair classified by Tstat. 
Only TCP flows generated by Instant Messaging applications are logged here.

Values are separated by spaces.

Each line is composed of two parts: values referring to the "client"
host (i.e. the host source of the first segment for the flow), 
and values referring to the "server" host (i.e. the destination of the first
segment of the flow).

For each Instant Messaging flow which is successfully tracked, a line in the log_chat_complete file
 will be written when the flow ends.
Here it follows a brief description of the columns.

###########################################################################################
#Col n# Short descr     # Long description                                 		  #
###########################################################################################
#  1  # Client IP addr  # IP address of the 'client' 
#  2  # Client port     # TCP port address for the 'client'
#  3  # Flow Size       # Flow Size [Bytes]
#  4  # Total packets   # No. of Total flow packets
#  5  # Total messages  # No. of Total messages sent by client
#  6  #	MSG_A		# No. of MSG_A sent by client [for MSN only, 0 for the others]
#  7  #	MSG_D		# No. of MSG_D sent by client [for MSN only, 0 for the others]
#  8  #	MSG_N		# No. of MSG_N sent by client [for MSN only, 0 for the others]
#  9  #	MSG_U		# No. of MSG_U sent by client [for MSN only, 0 for the others]
# 10  #	MSG_Y		# No. of MSG_Y sent by client [for MSN only, 0 for the others]
###########################################################################################
# 11  # Server IP addr  # IP address of the 'server' 
# 12  # Server port     # TCP port address for the 'server'
# 13  # Flow Size       # Flow Size [Bytes]
# 14  # Total packets   # No. of Total flow packets
# 15  # Total messages  # No. of Total messages sent by server
# 16  #	MSG_A		# No. of MSG_A sent by server [for MSN only, 0 for the others]
# 17  #	MSG_D		# No. of MSG_D sent by server [for MSN only, 0 for the others]
# 18  #	MSG_N		# No. of MSG_N sent by server [for MSN only, 0 for the others]
# 19  #	MSG_U		# No. of MSG_U sent by server [for MSN only, 0 for the others]
# 20  #	MSG_Y		# No. of MSG_Y sent by server [for MSN only, 0 for the others]
###########################################################################################
# 21  # Start Time	# Flow Start Time [in Unix Epoch Time]
# 22  # End Time	# Flow End Time [s]
# 23  # Chat Flow Type  # Chat Flow Type: see below
# 24  # Chat Version	# Version of the protocol used by the Instant Messaging application
# 25  # Internal	# Client IP address is internal ? (0=no, 1=yes)
# 26  # TCP Flow No.	# TCP Flow ID Number
# 27  # "T"		# Label to state a TCP Flow
# 28  # Chat Protocol	# Type of Upper Level Protocol [32=MSN, 64=Yahoo, 128=Jabber/GTalk]
###########################################################################################
